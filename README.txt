READ ME:

Steps to Compile the code.

1) Go inside make folder
2) Enter the command "make clean" in bash terminal, to delete all the object files and exe files.
3) Enter the command "make" in bash terminal, to compile and generate Client.exe
4) Enter the command "make -f server_make" in bash terminal, to compile and generate server.exe
5) Go inside the bin folder to find the generated exes.

Steps to Execute the exes.

1) Open 2 bash terminals
2) In one Bash terminal run the server.exe as " ./server.exe"
3) In another Bash terminal run the client.exe as " ./client.exe <server-Ip address> <Port No used by the server>
